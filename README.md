# Pingpong

## De qué trata?

Es una aplicación para ir llevando los resultados de los partidos de ping pong.
Para comenzar a jugar los jugadores deben crearse una cuenta y luego comenzar la partida ingresando sus mails.

No es necesario estar loggeado para jugar. Lo que se puede ver al estar loggeado son todas tus partidas. Si no lo estás vas a ver todas las partidas que se han jugado en la app.


#### Utilizar la app

1. Debes clonarla,
2. Utilizar el comando 'npm i' tanto en la carpeta api como en la carpeta client
3. Crear un .env en la carpeta client con las variables: 

```js
DB_USER
DB_PASSWORD
DB_HOST
PORT
DB_NAME
JWT_SECRET
API_URL
CLIENT_URL
```

4. Para correr api y client utiliza el comando 'npm start'
5. Para ver los test de api utiliza 'npm test'
